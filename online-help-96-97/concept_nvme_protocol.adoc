= NVMe傳輸協定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用System Manager來設定NVMe傳輸協定。NVMe是一種傳輸傳輸傳輸傳輸協定、可高速存取Flash型網路儲存設備。使用NVMe傳輸協定的系統有一個子系統、包含特定的NVMe控制器、命名空間、非揮發性儲存媒體、主機、連接埠、以及控制器與儲存媒體之間的介面。
